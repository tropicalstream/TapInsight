TapInsight Companion

Connect to your RayNeo X3 Pro glasses to configure the AI assistant, manage TapRadio stations, and more.

Connect via USB

Fastest and most reliable. Requires the glasses connected via USB cable with ADB debugging enabled.

1
Connect glasses via USB and run this once in your terminal:
adb forward tcp:19110 tcp:19110
Or run the helper: ./setup_usb.sh --watch for auto-reconnect
2
Click the button below. If the forwarding is active, it connects instantly.
Forwarding keeps dropping? Run ./setup_usb.sh --watch from the project folder. It automatically re-establishes the port forward whenever the glasses reconnect.

Connect via WiFi

Requirement: Your computer and glasses must be on the same WiFi network. On the glasses, go to Settings → WiFi → tap your connected network to see the IP.

Common LAN IP ranges: 192.168.0.x, 192.168.1.x, 10.0.0.x, 172.16.x.x
Find this in your glasses' WiFi settings. The companion server runs on port 19110.

Step 2 — Quick Setup Guide

1
Install the TapInsight APK on your RayNeo X3 Pro via ADB:
adb install tapinsight-debug.apk
2
Launch TapInsight on the glasses. The companion server starts automatically on port 19110.
3
Enter the glasses' IP above and click Connect. This opens the full companion app where you can configure everything.
4
In the companion app Setup tab, enter your Gemini API key (free from Google AI Studio).
5
Start talking! Tap the glasses touchpad to activate the AI assistant. It can see through the camera and respond by voice.

What You Can Configure

Once connected, the companion app has four tabs:

Setup

Gemini API key, AI model selection, Google OAuth (for Maps/Calendar), Spotify credentials, system prompt customization, HUD display settings.

Browser

Manage bookmarks and browser settings for TapBrowser on the glasses.

Dashboard

Customize the TapBrowser homepage — rearrange, add, or remove quick-launch links.

TapRadio

Manage internet radio stations, search 30,000+ stations online, toggle favorites, and sync to the glasses.

Troubleshooting

USB: localhost not working? The ADB port forward resets when the glasses disconnect or reboot. Re-run adb forward tcp:19110 tcp:19110, or use ./setup_usb.sh --watch for automatic reconnection.

WiFi: Can't connect? Make sure TapInsight is running on the glasses and both devices are on the same WiFi. Try ping YOUR_GLASSES_IP from your terminal.

Wrong IP? Your router may assign a different IP. Check your glasses' WiFi settings or your router's connected devices list.

Firewall? Ensure port 19110 isn't blocked by your network firewall.
